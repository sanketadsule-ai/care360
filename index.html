Configure Google OAuth Client

To connect your real Gmail inbox directly from localhost, you need a Google OAuth Client ID.

How to get a Google Client ID:

  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Go to **APIs & Services → Library**, search for **Gmail API**, and click **Enable**.
  4. Go to **APIs & Services → OAuth consent screen**, configure it for "External" users, and add the scopes `https://www.googleapis.com/auth/gmail.readonly` and `https://www.googleapis.com/auth/gmail.send`.
  5. Go to **APIs & Services → Credentials**, click **Create Credentials → OAuth client ID**.
  6. Select Application type: **Web application**.
  7. Under **Authorized JavaScript origins**, add: `http://localhost:8080` (and `http://localhost` if needed).
  8. Click **Create**, copy the **Client ID**, and paste it below.
